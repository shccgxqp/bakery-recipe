## 2. 單一圓形、方形與特殊造型蛋糕模具

單體蛋糕模具的幾何學焦點在於「邊緣應力分佈」與「脫模面摩擦力」的控制。學廚在此類別中採用活動底盤（Loose Bottom / Removable Bottom）與固定底盤（Ever Solid）雙軌技術路線。

對於戚風蛋糕（Chiffon Cake）這類高度依賴模具壁提供攀爬阻力、不允許使用不沾塗層阻礙其多孔結構發育的品類，學廚特別採用輕質高導熱的陽極氧化鋁合金（如WK9418）；而對於乳酪起司蛋糕（Cheesecake）、巴斯克（Basque）或重油海綿蛋糕，則使用高剛性碳鋼基材，搭配具備高延展密封度的活動式底盤，在防漏液與完美脫模之間取得平衡。

中空煙囪模（Tube Pan）的幾何中心柱能顯著縮短核心麵糊的受熱徑長，在高溫烘烤時形成空氣對流通道，使蛋糕體由內而外同步成熟，避免了外緣過焦而中心仍呈濕黏或凹陷的物理缺陷。此外，學廚與三麗鷗（Sanrio）聯名推出的 Hello Kitty 與 Cinnamoroll 造型模具，則展現了精密壓鑄與多曲面導熱應力控制的尖端工藝。

## 3. 多連點心與造型烤模

多連烤模（Multi-cavity Molds）展現了學廚在精密沖壓與表面物理塗層技術上的深度集成。針對法式傳統西點如可麗露（Canelé）及馬德蓮（Madeleine），學廚全面採用 0.8 毫米超厚重型碳鋼（如WK9864、WK9874），這在多連板模具中屬於極高配置。超厚碳鋼能提供極大的熱容，確保烤箱在開啟、置入模具的物理瞬間，模具本體的溫度跌幅降至最低，從而為可麗露的「蜂蠟高溫焦糖化」提供持續穩定的熱動能，烘烤出外殼深焦脆、內裡呈完美蜂巢狀的傳統法式質地。

其馬德蓮模具的貝殼紋理深邃，邊緣強化捲邊不僅能防止熱膨脹不均導致的鋼板扭曲，還能在高速熱風對流中，引導熱氣流越過模具表面，促使馬德蓮背面形成完美的「小山丘（Signature Hump）」。

## 4. 派、塔與菊花烤模

法式塔派（Tart / Pie）的成敗在於派皮（Pâte Sablée / Brisée）的油脂酥化程度與底部水分的瞬間汽化。學廚派盤的核心設計在於其「活底結構」與「精細菊花波浪邊」。活底設計能使烘焙完成後，透過下方的向上推力，在毫無摩擦力損害的情況下，將薄脆的派皮與側邊分流，徹底解決了黏壁崩塌問題。

在熱能傳遞上，學廚利用重型碳鋼的高效傳導熱性能，使塔皮底部的黃油在烤箱高溫下迅速沸騰，與麵粉纖維發生物理隔阻，從而形成多層次的酥鬆口感。部分進階款式（如9.5吋帶孔塔盤）更在底盤增設了精密氣孔，允許蒸汽直接向下宣洩，避免了盲烤時底皮因熱氣膨脹起泡的困擾。

## 5. 平板烤盤、冷卻網與烤箱配件

在烘焙馬卡龍（Macaron）、曲奇餅乾（Cookies）或平板蛋糕捲（Jelly Roll）時，邊緣高度與平整度是決定空氣對流分流的決定性因素。學廚平盤類模具採用低斜邊或無斜邊（Rimless）的設計，能顯著減少熱空氣在盤面邊緣產生的渦流阻滯，使高溫熱風能水平掠過餅乾表面，達到完全對稱的受熱著色效果。

在此類別中，學廚特別針對蛋糕捲、瑞士捲的特定工藝需求推出了兩款 11 吋正方形明星商品：水果捲模 WK9055 與 深底捲邊烤盤/奶凍捲模 WK9076。
* **WK9055** 採用鋁合金材質，其物理特性導熱極快，能讓輕薄的瑞士捲或水果捲蛋糕體在進爐後迅速定型並鎖住水分，防止蛋糕因過度烘乾而開裂。
* **WK9076** 則由厚重碳鋼製成，蓄熱能力強，且高度加深至 3.5 cm 並帶有加固捲邊，除了適合厚實多汁的奶凍捲蛋糕底外，也非常適合製作牛軋糖、布朗尼等高密度點心。
