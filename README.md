# 焙啾啾(烘焙帳本)🥐

這是我的**個人甜點食譜集 + 烘焙材料資料庫**。喜歡烤東西,也喜歡把每道甜點的
配方、成本、營養、過敏原算得清清楚楚,所以做了這本帳。

👉 **線上翻閱:<https://shccgxqp.github.io/bakery-recipe/>**

## 為什麼做這個(這個網站的宗旨,一直都是)

有人烘焙是為了謀生,有人是興趣,有人想把喜悅分享給別人,有人單純享受秤重、
反覆微調配方的實驗過程——這些都是很好的理由。

希望大家能慢慢感受烘焙的美好:出爐那一刻的喜悅,每次微調配方朝喜歡的口感
靠近一點,都值得珍惜。**焙啾啾**取自「烘焙久久」——希望烘焙是人生裡一個
能長久陪伴的小興趣。累了就休息,想烘焙的時候,選一道喜歡的甜點,回來就好。

## 📖 食譜:個人製作紀錄

食譜屬於**個人紀錄性質**——記錄我實際做過的甜點:配方用量、實作步驟的心得調整、
烤溫時間、成本與定價。**這些配方多數源自網路上的優秀創作者,並非我的原創**;
每道食譜盡量附上原出處連結(YouTube、部落格),想學做法請務必去看原作者的完整教學,
這裡只是我做過之後的量化紀錄。

每道食譜可以看到:

- 配方明細(含分層,例如奶油棒的餅乾層/奶油層)與參考原出處
- 材料成本、每份成本、利潤率試算
- 台式營養標示(每份 / 每 100 公克,有實秤成品重的以成品重計算)
- 過敏原資訊(由材料自動彙整)

## 🧈 材料資料庫:營養與過敏原資訊

材料主檔的目標是做成一份**公開、可查詢的烘焙材料安全資訊庫**。每項材料記錄:

- **過敏原標示**:依台灣「食品過敏原標示規定」的 11 類強制標示項目
  (甲殼類、芒果、花生、牛奶羊奶、蛋、堅果、芝麻、含麩質之穀物、大豆、魚類、
  亞硫酸鹽),區分「含有」與「可能含有(產線交叉污染)」
- **營養成分**:每 100g 的熱量、蛋白質、脂肪、飽和脂肪、反式脂肪、碳水化合物、
  糖、鈉——即台灣「包裝食品營養標示應遵行事項」的 8 項必要標示
- **成分**:照產品包裝「成分」欄原文記錄(含食品添加物的合規寫法)
- **廠牌與規格**:同一種材料、不同廠牌的成分與營養並不相同;
  每筆資料附**標示登記日**,標明數值是哪一天從包裝抄錄的(廠商會改配方!)

> ⚠️ 這些資料是我從產品包裝逐筆抄錄的個人紀錄,僅供參考。
> 廠商隨時可能更改配方,**實際過敏原與營養資訊請一律以你手上產品的包裝標示為準**;
> 對外販售的正式標示,依法應以實際檢驗或供應商文件為準,不可直接使用本站估算值。
