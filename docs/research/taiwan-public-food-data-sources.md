# 台灣公開食品資料來源地圖

> 建立：2026-07-15。目的：指定 `ingredients.evidence` 的可用來源，供材料研究與
> 第二次複查使用。這份文件不提供任何單一材料的可直接寫入數值。

## 結論先行

台灣有權威的**通用食品營養資料庫**，但目前找不到可公開查詢、可涵蓋所有
「市售品牌／包裝規格」營養標示的政府總資料庫。政府資料庫與品牌包裝不是可以
互相替代的兩類資料：前者適合原型食材，後者必須以原廠、台灣代理商或實際包裝
確認。

## 可直接採用的來源

| 來源 | 適用材料 | 可記錄的 evidence 類型 | 不能做什麼 |
| --- | --- | --- | --- |
| [衛福部食藥署：台灣地區食品營養成分資料庫](https://www.fda.gov.tw/tc/siteList.aspx?sid=284) | 原型食材與未指定品牌的通用食品：水、全蛋/蛋黃/蛋白、香蕉、草莓、藍莓、柳丁、核桃、杏仁、鹽、肉桂等 | `official_db` | 不可用來取代特定品牌奶油、鮮奶油、餅乾、糖漿、乳酪或添加物的包裝標示。 |
| [衛福部食藥署：台灣地區油脂資料庫入口](https://www.fda.gov.tw/tc/site.aspx?sid=271) | 油脂、奶油、食用油的通用脂肪酸資料比對 | `official_db` | 不是某一包市售油品的配方／品牌資料。 |
| [農業部產銷履歷農產品資訊網](https://taft.moa.gov.tw/) | 有追溯碼的生鮮農產品；可補產地與生產者資料 | `official_db`（營養值仍標示來源為 TFDA） | 不能作為一般進口水果、加工材料或市售品牌食品的營養標示。 |
| [衛福部食品核可資料查詢說明](https://www.mohw.gov.tw/cp-88-219-1.html) | 特殊營養食品、健康食品、特定需查驗登記食品的核可資料 | 視個案記 `manufacturer` 或 `official_db` | 一般食品通常不需事先查驗登記，因此不是一般烘焙材料的品牌查詢庫。 |

## 為何不能用政府資料庫補所有品牌商品

食藥署資料庫是食品成分樣品資料庫；官方研究資料說明它涵蓋多個食品類別與大量
營養素，是營養教育、食品標示計算與研究的重要基礎，但樣品不等於每一個市售包裝。
衛福部對「台灣市售商品營養標示資料庫」的公開資料建議回應也指出，市售商品的
配方與添加物比例不同，不能由通用資料庫的樣品名稱可靠推得特定商品營養值。

- [TFDA 食品成分資料庫的發展與應用](https://www.fda.gov.tw/tc/includes/GetFile.ashx?id=f637744876252406610)
- [政府資料開放平台：市售商品營養標示資料庫建議與機關回應](https://www.mohw.gov.tw/dl-91420-d6888c74-0325-45c7-9c0e-f2930d6b652a.html)

所以資料來源優先序固定為：

1. **同一包裝的實體標示／條碼**（最優先）。
2. **台灣進口商或總代理的現行規格書**。
3. **原廠的同規格、同市場產品頁或規格書**。
4. **可靠零售頁轉錄的包裝標示**（只能當候選，須二次核對）。
5. **TFDA 通用食品營養資料庫**（僅原型食材或無品牌通用材料）。

## 寫入規則

### 原型食材

```js
{
  brand: '',
  spec: '',
  verification: { status: 'verified', latestVerifiedAt: 'YYYY-MM-DD' },
  evidence: [{
    type: 'official_db',
    title: '台灣地區食品營養成分資料庫：<官方品項名稱>',
    url: '<對應查詢頁>',
    reference: '<官方食品編號或完整品項名>',
    checkedAt: 'YYYY-MM-DD',
    confidence: 'high'
  }]
}
```

### 包裝商品

資料來源必須能識別至少一項：條碼、淨重／容量、完整品名、產品料號或台灣進口商。
若只知道「某品牌鮮奶油 35%」，但不知道 200ml、1L、台灣／海外市場或產品線，
一律維持 `pending`。

## 研究順序

1. 先用 TFDA 補：水、蛋類、香蕉、草莓、藍莓、柳丁、核桃、杏仁、鹽、肉桂。
2. 再以台灣包裝來源補：奶油／鮮奶油／乳酪／餅乾／麵粉／糖漿等品牌材料。
3. 色膏、奶粉、吉利丁粉、葡萄糖漿、茶粉、可可粉、白鈕扣巧克力、棉花糖、希臘式優格等，
   先索取實物包裝或條碼；不要用名稱猜品牌。
