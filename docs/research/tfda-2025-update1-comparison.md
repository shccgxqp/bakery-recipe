# TFDA 2025 Update 1 Excel 與公開 API 核對

查核日：2026-07-15  
狀態：研究紀錄；**不寫入 MongoDB，不覆寫任何材料。**

## 檔案與比對方法

- 使用者提供的本機檔：`食品營養成分資料庫2025版UPDATE1EXCEL(另開新視窗).xlsx`。
- 活頁簿名稱為「台灣食品成分表」；資料列明示全部數值為「每 100g 可食部分」。
- 線上對照來源：[衛福部食藥署食品營養成分資料集（InfoId=20）](https://data.fda.gov.tw/opendata/exportDataList.do?InfoId=20&logType=5&method=ExportData)，以及其[政府資料開放頁](https://data.gov.tw/dataset/8543)。
- 用「整合編號」逐筆比對，而非只以中文名稱比對。Excel 保存的是高精度分析值；公開 API 的 `每100克含量` 為顯示用四捨五入值。

## 結果：Excel 與食藥署公開 API 一致

下列 7 個樣品、共 49 個核心營養欄位（熱量、粗蛋白、粗脂肪、飽和脂肪、總碳水、糖質總量、鈉）均一致；API 僅做位數圓整。例如草莓 `38.6848 kcal` 在 API 顯示為 `39 kcal`，雞蛋黃平均值 `307.9919 kcal` 顯示為 `308 kcal`。

| 整合編號 | Excel 樣品 | Excel 的前處理／樣品條件 | API 結果 |
| --- | --- | --- | --- |
| C0405101 | 生核桃 | 生、混合均勻打碎 | 一致（顯示圓整） |
| D1400101 | 草莓 | 去蒂、混合均勻打碎 | 一致（顯示圓整） |
| D3600101 | 柳橙 | 普遍系；去皮及籽、混合均勻打碎 | 一致（顯示圓整） |
| D5000101 | 藍莓（台灣） | 冷藏盒裝、混合均勻打碎 | 一致（顯示圓整） |
| K01081 | 雞蛋黃平均值 | 生；去殼及蛋白、混合均質 | 一致（顯示圓整） |
| P0100601 | 肉桂粉 | 混合均勻（肉桂粉、清華桂粉等） | 一致（顯示圓整） |
| P0300101 | 岩鹽 | 混合均勻（喜馬拉雅山岩鹽） | 一致（顯示圓整） |

因此，後續匯入原型食材時，可在「來源快照」保存 Excel 的精確值與整合編號；使用者介面則依台灣營養標示的呈現規則另外處理小數位，不能把 API 顯示圓整值當作新的分析原始值。

## 尚未找到安全對應項目

本次 Excel 中沒有找到可直接、無歧義對應現有「香蕉」、「水」、「杏仁粒」或一般「鹽」的同名原型樣品。這不是缺資料就改採其他國家資料庫的理由；維持現有值／待查，直到可指定 TFDA 整合編號或取得適用包裝資料。

## 寫入前置條件

依 `ingredient-versioning-decision.md` 的決策，以上所有「待建立新版」項目都要等 `ingredientVersions` 與食譜 `ingredientVersionId` 的不可變綁定完成後才能寫入。完成後每筆至少保存：`sourceType: official_db`、整合編號、Excel 檔案版本、前處理描述、擷取日與精確每 100g 數據。
