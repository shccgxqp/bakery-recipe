# 烘焙帳本 🥐

這是我的**個人甜點食譜集**。純粹興趣使然——喜歡烤東西,也喜歡把每道甜點的
配方、成本、營養算得清清楚楚,所以做了這本帳。

**一切公開**:每道食譜的完整配方、材料用量、成本結構、營養標示都在這裡,
歡迎自由取用、照著做、改著做。

👉 **線上翻閱:<https://shccgxqp.github.io/bakery-recipe/>**

## 目前收錄

提拉米蘇、起司蛋糕、可爾必思生乳捲、柳丁磅蛋糕、香蕉磅蛋糕、焙茶戚風蛋糕、
焦糖伯爵紅茶蛋糕、黑糖奶油棒、原味奶油棒、厚燒奶油酥餅、雪球餅乾、莎布蕾、
雪Q餅、馬卡龍、可麗露、馬琳糖、費南雪、司康……持續增加中。

每道都能看到:

- 配方明細(含分層,例如奶油棒的餅乾層/奶油層)
- 材料成本與每份成本
- 台式營養標示(每份 / 每 100 公克)
