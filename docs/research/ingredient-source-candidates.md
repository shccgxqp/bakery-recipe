# 材料資料來源候選清單

> 建立：2026-07-15。用途：供第二位審核者逐筆複查，**不是可直接寫入資料庫的資料檔**。
> 每筆只有在「產品完全相同、規格相同、台灣包裝（或原廠現行規格）一致」後，才可
> 寫入 `ingredients`。任何差異均維持 `verification.status:"pending"`。

## 查核規則

1. 原廠／台灣代理商／進口商的產品規格或包裝標示優先。
2. 零售頁可作為「發現候選與抄錄台灣標示」的輔助，但不可單獨視為最終證據。
3. 原型食材另用衛福部食品營養成分資料庫；不套用到指定品牌包裝商品。
4. 一個來源只要與目前數值有一項不一致，就不自動更新；需查包裝條碼、規格或實物。

## Claude 複查工作單

- 檢查每個網址是否仍是原廠／台灣進口商的現行資料，並取得頁面或包裝的查核日期。
- 對「中」信心的零售頁，尋找同一包裝條碼的原廠、代理商或實拍佐證。
- 不可用其他市場或不同淨重／配方的產品填入現有材料。
- 確認後才產出可寫入的 patch：`brand`、`spec`、`per100g`、`allergens`、
  `mayContain`、`subIngredients`、`verification`、`evidence`；不要改 `_id`。
- 本 batch 目前**沒有任何一筆獲准寫入 MongoDB**。

## Claude 複查結果（2026-07-15，第一輪：核對文件原引用網址）

逐筆重新開啟文件內列出的網址核對：

1. **依思尼無鹽奶油** — 原引用網址（tsaifongfood）已 **404**，頁面已下架。轉錄內容無法對照現存頁面重新驗證。
2. **水手牌高筋麵粉** — 確認 `lhmilling.com/flour-detail/F1506__1/` 為「水手牌**無添加**高筋麵粉」（蛋白質 12.6～13.9%，無完整營養標示表）；PChome 頁為「水手牌**特級強力粉**《高筋》」，營養標示（350kcal／12.3g蛋白／1.1g脂肪／0.2g飽和／72.7g碳水／0g糖／1mg鈉）與過敏原文字皆與文件轉錄相符。**兩者確為不同品項**，文件原標注「需先確認實物」成立，維持待查。
3. **維西尼手指餅乾** — 官方頁確認 381kcal／7.8g蛋白／3.8g脂肪／0.9g飽和／78g碳水／42g糖／0.21g鹽，成分與過敏原（小麥、蛋；可能含大豆、芥末）皆與文件轉錄相符。**確認碳水官方78g、現有資料庫80g有落差**，needs_review 標記成立。

5. **安佳動物鮮奶油** — 官方頁（`anchorfoodprofessionals.com/sg`）確認 334kcal／2.0g蛋白／35.5g脂肪／23.3g飽和／1.0g反式／2.9g碳水糖／21mg鈉，1L，成分「Cream＋Stabiliser(407)」。此為**新加坡站版本**，與現有資料庫（342kcal等）數值不同，證實「地區／產品未鎖定」疑慮成立。

## Claude 複查結果（2026-07-15，第二輪：獨立搜尋，不依賴文件原網址）

不引用文件已列來源，改自行搜尋比對，發現兩項需要更新文件認知的新事實：

1. **依思尼無鹽奶油**：搜尋多家台灣經銷商頁（champion-hands、ycbake、finebaking、madamruby等），營養標示均以圖檔呈現，**找不到第二個文字來源佐證**原轉錄數字。信心未提升，仍待實物或進口商一手文字頁確認。
2. **水手牌高筋麵粉（新發現）**：獨立查到聯華官方「特級強力粉(高筋)」頁 `lhmilling.com/flour-detail/F1501/`，確認：
   - 蛋白質 12.0～13.0%（與PChome轉錄12.3g相符）
   - **成分含多種添加物**：小麥澱粉、樹薯澱粉、澱粉液化酵素、木聚醣酶、脂肪酶、半纖維素酶、葡萄糖氧化酶、糖化酵素、維生素C、糊精、磷酸鈣、鹽 — **非單純小麥粉**
   - 與「無添加高筋麵粉」(F1506，蛋白質12.6～13.9%，成分無添加物) 為明確不同品項
   → 若未來確認實物為「特級強力粉」，patch 的 `subIngredients` 需完整列出上述添加物，不能只抄零售頁簡化版本。
3. **維西尼手指餅乾**：第三方彙整（lecalorie.it 等）給出 379kcal／碳水77.5g，與官方頁381kcal/78g有小數點級落差，但**再次確認碳水落在77~78g區間、非現有資料庫80g**，強化 needs_review。
4. **愛樂微35%鮮奶油**：OpenFoodFacts／FatSecret 等第三方資料庫交叉確認與官方頁一致（335kcal等），無新增疑慮。
5. **安佳動物鮮奶油（新發現）**：找到台灣實際進口商頁 `newyoung.com.tw`（新加坡商永紐股份有限公司台灣分公司），成分為「乳脂、乳固形物、關華豆膠、乳化劑(脂肪酸甘油酯、聚山梨醇酯80)、增稠劑(玉米糖漿固形物、鹿角菜膠)」——**與文件引用的新加坡官網版本成分（僅Cream+Stabiliser 407）明顯不同**。
   → 確認文件引用來源（Anchor SG官網）**不是台灣實際在賣的產品版本**；台灣進口版配方更複雜。頁面未附文字版營養標示數字（用圖檔），仍需另外取得台灣版本的實際營養數字，不可沿用SG官網數字。
6. **菲力鮮奶油乳酪**：查到美版「Original cream cheese spread」第三方資料庫約258kcal/100g，與AU產/TW進口PDF的224kcal、現有資料庫308kcal**三個數字互不相同**，確認至少三種區域配方並存，強化「現有資料庫並非同一品項」的高信心判斷。
